# SeniorBrowse Store Assets

Place your 5 Chrome Web Store screenshots here (1280×800 each):

1. **screenshot-1-hero.png** — Featured/hero image with main value prop
2. **screenshot-2-home.png** — Home screen with shortcuts and side panel
3. **screenshot-3-sidepanel.png** — Side panel tour
4. **screenshot-4-settings.png** — Settings modal (caregiver admin view)
5. **screenshot-5-block.png** — Blocked ad site view (optional but recommended)

## Screenshot Specs

- **Size:** 1280×800 pixels (required)
- **Format:** PNG
- **Max file size:** 5 MB total for all screenshots
- **Quality:** Clear, readable text and UI

## How to Capture

1. Open Chrome with SeniorBrowse extension loaded
2. Open Developer Tools (F12)
3. Ctrl+Shift+M to toggle device mode
4. Set viewport to 1280×800
5. Screenshot each screen:
   - New tab (home)
   - Any website with side panel visible
   - Settings modal (opened via Edit mode)
   - Activity log tab
   - Any other key feature
