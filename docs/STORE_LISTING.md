### Long Description

```
The browser extension that makes the internet safe and easy for your loved ones — while giving you full caregiver control behind the scenes.

Does your parent or grandparent struggle with confusing websites, scary pop-ups, or accidentally clicking the wrong thing? SeniorBrowse turns browser into a calm, simple, and safe experience built just for them — no tech knowledge required.

WHAT THEY SEE
• A warm, personal home screen with big, easy-to-read shortcut tiles
• A helper panel on every webpage with one-tap buttons: Home, Back, Text Size, Save Page
• Large, readable text that can be increased with a single click
• A calm, focused interface with no clutter or distractions

WHAT YOU GET (as caregiver)
✓ Full control behind the scenes with PIN-protected settings
✓ Safety on autopilot — dangerous websites blocked, downloads controlled, ads eliminated
✓ Activity log to see every page visited, search made, and page saved
✓ Complete customization — add/remove shortcuts, rename buttons, adjust text size defaults
✓ Guided tour that can be restarted anytime

INVISIBLE PROTECTION
✓ Known malware and phishing sites blocked automatically
✓ Downloads disabled to prevent accidental software installation
✓ Ad blocking removes flashing banners and misleading buttons
✓ Your own custom block list for additional control

HOW IT WORKS
1. You complete the 5-minute setup wizard (names, shortcuts, safety settings)
2. Your loved one gets a friendly guided tour of all features
3. They browse with confidence every day while you stay informed

COMPLETELY FREE
No account, no trial, no subscription. Install once and every feature is available immediately.

Questions? Contact kamilpinas@gmail.com
```

## How to Add Screenshots

1. **Take real screenshots** of the extension running in Chrome:
   - Install extension in Chrome
   - Open a new tab for the home screen
   - Navigate to a website to show the side panel
   - Open Settings modal for the admin view
   - Capture each at 1280×800 resolution

2. **Save as:**
   - `public/store/screenshot-1-hero.png`
   - `public/store/screenshot-2-home.png`
   - `public/store/screenshot-3-sidepanel.png`
   - `public/store/screenshot-4-settings.png`
   - `public/store/screenshot-5-activitylog.png` (optional)

3. **Create folder if needed:**

   ```bash
   mkdir -p public/store/
   ```

4. **Optimize:** Compress to < 5MB total (Chrome Store requirement)
